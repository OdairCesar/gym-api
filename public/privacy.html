1. Quem somos

A Gym API é uma plataforma de software no modelo SaaS (Software as a Service) voltada para a gestão de academias. Por meio de uma API RESTful multi-tenant, oferecemos funcionalidades de gerenciamento de usuários, dietas, treinos e produtos para academias ("Clientes") e seus respectivos membros ("Usuários Finais").

Nesta política, "nós" refere-se ao time responsável pela operação da Gym API. "Você" refere-se tanto aos administradores das academias quanto aos usuários finais que acessam a plataforma através de integrações construídas sobre nossa API.

2. Dados que coletamos

2.1 Dados fornecidos diretamente

Durante o registro e uso da plataforma, coletamos as seguintes categorias de dados:

  • Dados de identificação: nome completo, e-mail, CPF
  • Dados de contato: telefone, endereço
  • Dados demográficos: data de nascimento, gênero
  • Dados profissionais: profissão (para personals/coaches)
  • Dados de saúde e atividade física: dietas, refeições, alimentos, treinos, exercícios, carga e repetições
  • Dados de acesso: tokens de autenticação, data/hora de login

2.2 Dados coletados automaticamente

  • Endereço IP e User-Agent nas requisições à API
  • Logs de acesso (audit trail) para fins de segurança
  • Metadados de operações (timestamps de criação e atualização de registros)
Dados sensíveis: Informações sobre dietas e atividades físicas podem ser consideradas dados sensíveis de saúde nos termos da LGPD (Lei nº 13.709/2018). Tratamos esses dados com camada adicional de proteção.

3. Como utilizamos os dados

Utilizamos os dados coletados para as seguintes finalidades:

  • Criar e gerenciar sua conta na plataforma
  • Autenticar e autorizar o acesso aos recursos da API
  • Possibilitar que personals/coaches acessem e editem planos dos seus alunos
  • Controlar limites de uso conforme o plano contratado pela academia
  • Garantir o isolamento de dados entre academias (multi-tenant)
  • Detectar e prevenir fraudes, abusos e acessos não autorizados
  • Prestar suporte técnico quando solicitado
  • Cumprir obrigações legais e regulatórias

Não utilizamos seus dados para fins de publicidade comportamental ou venda a terceiros.

5. Compartilhamento de dados

Seus dados podem ser compartilhados nas seguintes situações:

5.1 Dentro da plataforma (multi-tenant)

Dados de um usuário são acessíveis apenas dentro do contexto da academia à qual pertencem. Personals de outras academias somente acessam dados de um usuário mediante autorização explícita concedida pelo próprio usuário ou pelo admin da academia (permissões cross-tenant).

5.2 Provedores de pagamento

Para assinaturas pagas, dados de pagamento são transmitidos diretamente ao provedor (Google Pay / Apple Pay) e não são armazenados em nossos servidores. Seguimos o padrão PCI-DSS para essas integrações.

5.3 Obrigação legal

Podemos divulgar dados quando exigido por lei, ordem judicial ou autoridade governamental competente.

Não vendemos, alugamos ou trocamos dados pessoais com terceiros para fins comerciais.

6. Retenção e exclusão

Mantemos seus dados pelo tempo necessário para a prestação dos serviços contratados ou conforme exigido por lei.

  • Dados de conta ativa: retidos durante toda a vigência do contrato
  • Logs de acesso e segurança: retidos por até 12 meses
  • Dados após cancelamento: excluídos em até 90 dias após solicitação, salvo obrigação legal de retenção
  • Dados fiscais/financeiros: retidos pelo prazo legal obrigatório (5 anos)

7. Segurança

Adotamos medidas técnicas e organizacionais para proteger seus dados:

  • Senhas: criptografadas com algoritmo Scrypt (resistente a ataques de força bruta)
  • Autenticação: Bearer Tokens com expiração configurável; logout invalida o token imediatamente
  • Transporte: comunicação via HTTPS/TLS em produção
  • Banco de dados: queries parametrizadas via ORM, prevenindo SQL Injection
  • Isolamento: todas as queries filtradas por gym_id, impedindo acesso cross-tenant não autorizado
  • Rate limiting: proteção contra brute force e abuso da API
  • Monitoramento: rastreamento de erros e anomalias em tempo real

8. Seus direitos (LGPD)

Nos termos da Lei Geral de Proteção de Dados (Lei nº 13.709/2018), você possui os seguintes direitos:

  • Confirmação e acesso: saber se tratamos seus dados e acessá-los
  • Correção: solicitar correção de dados incompletos, inexatos ou desatualizados
  • Anonimização, bloqueio ou eliminação: quando o tratamento for desnecessário ou excessivo
  • Portabilidade: receber seus dados em formato estruturado e interoperável
  • Eliminação: excluir dados tratados com base no consentimento
  • Revogação do consentimento: retirar consentimento a qualquer momento
  • Informação: sobre as entidades com quem compartilhamos seus dados
  • Oposição: quando tratar com base em legítimo interesse
Para exercer qualquer um desses direitos, entre em contato através do canal indicado na seção 10. Responderemos sua solicitação em até 15 dias úteis.

9. Cookies e logs

A Gym API é uma API REST e, em sua forma nativa, não utiliza cookies. Logs de acesso (IP, endpoint, timestamp) são registrados para fins de segurança, diagnóstico e prevenção de abusos, conforme descrito na seção 2.2.

Aplicações de terceiros construídas sobre a Gym API podem fazer uso de cookies em suas interfaces — essas práticas são reguladas pelas políticas de privacidade de cada aplicação.
